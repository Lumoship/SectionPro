# SectionPro

Free section properties calculator for marine structural profiles.

## Features

- Supports L, T, Flat Bar (FB), and Bulb Flat (HP) profiles per EN 10067
- Attached plate calculations
- BV NR467 corrosion allowance
- Interactive cross-section sketch
- Instant calculations (no signup, no installation)

## Use Cases

- Ship structural design
- Hull scantling calculations
- Class society compliance checks (DNV, LR, BV, RINA)
- Educational and training purposes
